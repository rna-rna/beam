To implement single-click for selection and double-click for opening a gallery, we need to enhance the MainContentV2 gallery item behavior. Here's how to do it step by step:

Step 1: Update Click and Double-Click Logic
We can use setTimeout to distinguish between a single-click and a double-click. This ensures that single-click behavior won't interfere with double-clicks.

import React, { useState } from "react";
import { useQuery } from "@tanstack/react-query";

export const MainContentV2 = () => {
  const [selectedGalleries, setSelectedGalleries] = useState<string[]>([]);
  const { data: galleries = [], isLoading } = useQuery({
    queryKey: ["/api/galleries"],
    queryFn: async () => {
      const res = await fetch("/api/galleries");
      if (!res.ok) throw new Error("Failed to fetch galleries");
      return res.json();
    },
  });

  const handleSingleClick = (id: string) => {
    // Toggle selection on single click
    setSelectedGalleries((prev) =>
      prev.includes(id) ? prev.filter((gid) => gid !== id) : [...prev, id]
    );
  };

  const handleDoubleClick = (id: string) => {
    // Open gallery on double-click (navigate to gallery page)
    console.log(`Opening gallery ${id}`);
    window.location.href = `/gallery/${id}`; // Adjust this URL as needed
  };

  const handleClick = (id: string) => {
    let clickTimeout: NodeJS.Timeout | null = null;

    // If a click occurs, check for a double-click within 250ms
    if (clickTimeout) {
      clearTimeout(clickTimeout);
      clickTimeout = null;
      handleDoubleClick(id);
    } else {
      clickTimeout = setTimeout(() => {
        handleSingleClick(id);
        clickTimeout = null;
      }, 250); // Adjust delay if needed
    }
  };

  if (isLoading) {
    return <div>Loading...</div>;
  }

  return (
    <div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4 gap-6">
      {galleries.map((gallery) => (
        <div
          key={gallery.id}
          onClick={() => handleClick(gallery.id)}
          className={`group bg-white shadow-md rounded-lg overflow-hidden cursor-pointer ${
            selectedGalleries.includes(gallery.id) ? "border-2 border-blue-500" : ""
          }`}
        >
          <div className="aspect-w-4 aspect-h-3 bg-gray-200">
            <img
              src={gallery.thumbnailUrl || ""}
              alt={gallery.title}
              className="object-cover w-full h-full group-hover:scale-105 transition-transform"
            />
          </div>
          <div className="p-4">
            <h3 className="font-semibold text-lg">{gallery.title}</h3>
            <p className="text-sm text-gray-600">{gallery.imageCount} images</p>
          </div>
        </div>
      ))}
    </div>
  );
};
Step 2: Explanation of the Click Handling
Single Click for Selection:

Clicking a gallery toggles its selection state.
Selected galleries will have a visual indicator (e.g., a blue border) via the selectedGalleries state.
Double Click for Opening:

Double-clicking a gallery immediately navigates to its detail page using window.location.href or a routing function like setLocation.
Handling Click Timing:

A setTimeout with a 250ms delay detects whether the second click happens within this timeframe.
If the second click occurs within 250ms, the handleDoubleClick function is executed, overriding the single-click logic.
